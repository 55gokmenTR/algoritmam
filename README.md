# 🧠 Algoritmam

**Verimliliğinizi arttıran kişisel bir karar ağacı.**

## 🎯 Projenin Amacı

> "Boş kaldığım vakit irade savaşı verip kaybetme riskini göze almamak için - karar vermeyi dış kaynaklara bırakan - bir algoritma hazırladım."

Karar verme yorgunluğu (Decision Fatigue) yaşamamak ve zamanı verimli kullanmak için bu süreci otomatize ettim.

## ⚙️ Teknolojik Altyapı

Bu projede **Ruby** programlama dili kullanıldı.

* **Neden YAML?** Karar ağacının seçeneklerinin kolayca değiştirilebilmesi ve okunabilmesi için veri yapısı olarak YAML tercih edildi.
* **OOP Tasarımı:** Kodun sürdürülebilir olması için Nesne Yönelimli Programlama prensiplerine sadık kalındı.

## 🚀 Kurulum

Projeyi bilgisayarınıza indirmek için terminale şu komutları yazın:

```bash
git clone [https://github.com/55gokmenTR/algoritmam.git](https://github.com/55gokmenTR/algoritmam.git)
cd algoritmam
ruby main.rb
```
